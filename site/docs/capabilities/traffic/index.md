This section provides information about traffic routing related capabilities in Envoy AI Gateway.

## Available Features

- [Model Virtualization](./model-virtualization.md) - Abstract model names from underlying providers
- [Provider Fallback](./provider-fallback.md) - Automatic failover between AI providers
- [Usage-based Rate Limiting](./usage-based-ratelimiting.md) - Token-based rate limiting for LLM requests
- [Response Caching](./response-caching.md) - Cache LLM responses to reduce latency and costs
